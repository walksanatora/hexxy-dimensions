      咒法维度

      在看到那样的事物时。
      我便意识到，我能随我心意将其塑形。
      我只是需要正确的图案。
      我似乎还会遇到几种新事故，大多数都是因为维室未准备好、或是已不再存在。

      我不应大肆破坏周围的墙壁，不然可能会引起它们的注意，结果就是我被扔出维室。


      为操作这种力量，我设计了一种新 iota，叫做维室。
      维室 iota 似乎会给予我对某片超维空间强大的控制力，后页的法术就是创建和操纵这片空间的工具。需注意，除非标明媒质消耗量，就无需消耗媒质。

      为限制维室 iota 的操纵功能（比如说只给朋友部分权限），这种 iota 带有 3 种权限设置（R、W、X），默认均启用。R 权限允许进入维室，W 权限允许删除维室，X 权限允许我将施法范围转移到维室中。埃弗雷特之权限可以禁用这些权限。

      （再明确一下，你其实可以“破坏”墙壁，这么做后你会受到非致命伤害，并返回主世界。）

          按照给定的长宽高制造一个新维室。每方块消耗 6 个充能紫水晶。


          每次新造维室时，似乎必须先等其“雕琢”完毕。
          在自然雕琢完毕之前，我无法在其中做任何有实际效果的事。

          雕琢的速度约为每秒 20 个方块。可以用一个简单图案查看还有多少仍待雕琢。
